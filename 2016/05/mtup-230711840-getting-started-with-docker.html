Getting started with Docker

marți, 24 mai 2016

A meetup explaining how to use Docker for development that shows practical solutions to using it for complex applications and taking it from the local environment to a production server.

We will show you how to work with Docker, Docker Compose, Docker Machine, Kitematic, Nginx-Proxy image and how to setup different project types in Docker containers. You can bring your laptops and test everything in your own setup with your own code. We will show you how to install and configure everything locally and the steps you need to take to deploy online.

The meetup is open format so that all the members can join and share their experience.

Bring your laptop :)

Difficulty level: Medium
