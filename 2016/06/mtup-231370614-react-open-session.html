React Open Session

joi, 2 iunie 2016

Hello clujsers!

Really happy to announce a new meetup, this time focused around React.js and the Flux architecture. We will try to discuss in an open format , with no dedicated speakers, about different problems or solutions to problems we found while working with React in a production application or simply when checking it out as part of personal projects.

We don't have a fixed schedule for the meetup, but we will try to address at a minimum level the following topics:

• React project setup / build tools


• React Components


• Flux Architecture


• Server-side Rendering


• Scaling a React codebase

Feel free to suggest other topics in the comments or during the meetup. We really encourage everyone to contribute.

This meetup is addressed to everyone in the community who has basic understanding of the React ecosystem and wants to share his/her thoughts on it or just maybe grab some ideas from other people working with it.

If you have an interesting project you would like to share/demo, please bring your laptop and we will make sure you have time to show us what you did.

Meetup Level : intermediate
