Automation Testing using C# with Selenium & Unit Testing

marți, 28 iunie 2016

Automation Testing using C# with Selenium (RO, all levels)

The presentation will include a description of the UI Automation Testing Framework’s structure that is used for all Yardi Romania products.

We will talk about Page Object Pattern, a popular design in test automation that helps to enhance test maintenance, makes tests more readable and robust, and also reduces code duplication.

We will also cover specific aspects related to the “Continuous Integration” tool that we use, and a short demo of an automatic test written for one of our projects.

About the speaker:

Andrei Florea is QA Automation Engineer at Yardi Romania with over 5 years of experience in the IT field both on manual and automation testing. The projects he is most proud of are creating frameworks in C# using “WebAii Telerik Framework” and “Selenium WebDriver”.

***


Unit Testing (RO, entry level)

This presentation will be a short walk trough theoretical concepts of unit testing.

We will talk about unit tests, what is a mock, stub, fake. We will show a lists of used tools for unit tests and mocking both for C# and Java.

We will also cover test driven development what it is and what are the benefits of using it.

About the speaker:

Cornea Robert is a C# developer at RIA Solutions Group with over 7 years of experience in the IT field. His first contact with unit testing was 4 years ago when he used these concepts to develop a complex application with small margin for errors. His main goal is to share his experience with unit testing with fellow coders.
