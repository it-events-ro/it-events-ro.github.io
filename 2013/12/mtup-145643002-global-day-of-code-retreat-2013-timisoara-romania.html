A Little History

Romania was the first non-US country to host a code retreat, so we have a tradition and passion for this type of events. Timisoara has organised a Global Day of Code Retreat since the event begun 3 years ago.

Regardless if you are a pro or a novice in programming, you are welcome to join a full day event created by coders for coders. This is a great opportunity to play, explore and learn programming from others like you. The event is hosted in about 200 cities around the world simultaneously. Last year there were more than 2000 participants world-wide.

Event Structure

We have a simple(-ish) problem to solve in different ways. It's Conway's Game of Life.

- There will be 40 minutes sessions
- Each session will have some kind of fun limitation on what we are allowed or not to use in our programming
- After each session the code goes bye-bye and we start from scratch with new ideas

Your Benefits

So what can you learn at this event?

- Experience pair programming with strangers
- Experiment with whatever programming languages you want
- Write code as you wish not as your boss tells you how
- Meet other passionate programmers like you

Welcome and We are Expecting You

So if you wish to join for a full day or just for a couple of sessions, feel free to reserve a place. We have enough seats for everyone interested, so don't worry you will not be left out.
