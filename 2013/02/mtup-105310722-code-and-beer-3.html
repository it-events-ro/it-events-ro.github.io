Code and Beer #3

miercuri, 20 februarie 2013

Today was Radu Ticiu's birthday (send him a greeting via Facebook if you haven't already) and our colleagues at TiMo threw a party for him the other evening. As luck would have it, not all the beer disappeared -- very good news for us!

As a consequence, we are organizing an improptu Code and Beer meetup at the Startup Hub on Wednesday evening to help him empty the beer kegs.

So, bring your laptop and your thirst (for knowledge http://img2.meetupstatic.com/1723755200050492932723/img/smileys/biggrin.gif) and let's spend an evening coding. If you don't know the concept, you will learn about the coding kata format. We'll pair up and code the solution to a problem that the pair chooses from a catalog of various challenges. And, um... there might also be some drinking.

All languages are ok. The more exotic, the better.

See you all tomorrow evening!
