Coderetreat

sâmbătă, 6 iulie 2013

Why?

In our day to day jobs we have the sense of wanting to get it done, that causes to rush. And so we try to cut corners, but sometimes we do not cut the corners in the best way because we do not know enough.

You will have fun , meet other programmers , learn new things from the others and have a lot of interesting discussions with everyone.

How?

During a coderetreat we use pair-programming , which is wonderful because you can learn a lot from each other.

We want to focus on writing beautiful code, there is no pressure to actually produce a working solution for the problem. But you will be pressured by the facilitator to write the best code you can, given some clear code quality and code design constraints.

You will try to experience new things and experiment new ideas and concepts.

Format

Take a problem: Conway’s Game of Life During 45 minutes sessions write the best code with your pair Delete your code (yes, it can be hard) 10 minutes retrospectives 5 minutes brake Change your pair Start a new 45 minutes session This code retreat is language agnostic , the main principles are the same for each language.

This event will be facilitated by Adi Bolboaca
