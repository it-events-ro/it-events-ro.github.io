Code and Beer #1

joi, 22 noiembrie 2012

Spend an evening with fellow hackers.

This is a new type of event we are starting in Timisoara. Think of it as a 'social coding' meetup where programmers gather, have a couple of beers and write some code.

Things you could do:
- hack on a pet project or open source project
- try a new tool (MongoDb, Hadoop, Node.js etc.)
- try a new programming language
- pair on a coding kata ( http://codingdojo.org/cgi-bin/wiki.pl?KataCatalogue )
- discuss design or architectural problems
- do your coursera assignment
- just come, have a drink, socialize and have fun

For the first event, the kind people at Unified Post offered to host us and even provide us with snacks and belgian beer. For the next ones we'll probably hit the city's pubs.

Ideally, bring your laptop. If not, you can pair with someone.

Please consider your RSVP carefully and update it in case you can't make it anymore.
