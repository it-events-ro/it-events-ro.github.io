uberVU Hack Days #5

sâmbătă, 26 mai 2012

Cum aplicatiile web realtime devin tot mai populare, ne dorim sa lansam cel mai tare framework de JS opensource care sa permita modularizarea aplicatiilor si dezvoltarea rapida. Vrem sa repetam ce a facut Rails pentru web 2.0, dar pentru aplicatiile realtime de astazi. Frameworkul este super-modular, scris in CoffeeScript si bazat pe backbone.js.

Proiectul este incubat de uberVU, unul din cei mai importanti playeri de aplicatii social media pentru enterprise. Este primul nostru proiect open-source si cu ajutorul tau putem sa cream cea mai tare platforma de dezvoltare a aplicatiilor web de pana acum.

Teme:

Platforma este in mare parte gata, dar sunt multe imbuntatiri pe care le putem face la nivel de core: asset bundling, garbage collection samd. Lista completa o gasesti online: http://bit.ly/LaURuB
