Agile & Software craftsmanship - Code retreat

sâmbătă, 20 iunie 2009

Software craftsmanship - Code retreat#1
Software craftsmanship manifesto - "Not only individuals and interactions, but also a community of professionals"

Agile is about quality, results and learning. Software developers are proud of their code, but there is always something that can be done better (if we look in a honest way to our software applications). Code retreat is a fast-learning technique 'invented' by software craftsmanship movement (derived from XP). How we will work? We will solve a problem, in pairs, on the same computer, one hour iterations. After each hour the code is completely deleted and the solving is re-done. This time we will try with 3 iterations. The purpose is to learn&practice programming techniques.

What you have to bring: a laptop, your pair-for-programming, the programming-mood
Don't worry about the language, you&your pair can use any language you are comfortable with. Just be sure the laptop has the tools installed and functional for the chosen programming language.

We will share best practices, learn and practice together better programming techniques while being relaxed and having fun in a summer Saturday morning.
