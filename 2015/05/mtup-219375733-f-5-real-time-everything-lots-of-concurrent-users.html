F.5 - Real time everything & lots of concurrent users

joi, 28 mai 2015

In the large variety of operating systems, processing units and devices on todays market is hard to implement a cloud based solution that accomplish every functionality and runs on every device.

Think of that as joining together smart phones, computers, smart TV 's, IOT devices running mobile, embedded or desktop operating systems persistently connected to a cloud application served via web server proxy over HTTP as you keep resources at the lowest level. We need to transparently acquire, process, transport and deliver various data stream types (data, video, audio, etc.) over persistent connections.

For these reasons, now almost 11 years began Concept application server development. It is an application server that uses a bidirectional protocol based on sockets or native Web Sockets. Is backed up by a programming language - Concept Programming Language, a virtual highly efficient machine up to 20x faster than PHP 5.6 as memory consumption is up to 10 times smaller and Concept Framework, a JavaScript like syntax but strictly object-oriented. The framework covers over 7000 API, from system level functions up to high level protocols such as SIP.

The high efficiency of Concept is given by Gyro Gears application generator, capable of generating complex applications in minutes and covering areas from the database to communication systems.

Eduard Suică, core developer and owner at Devronium, is an pasionante paraglider pilot and software developer. We will meet him at Brașov Tech Meetup on Thursday at 18:30 to talk about his work and vision.
