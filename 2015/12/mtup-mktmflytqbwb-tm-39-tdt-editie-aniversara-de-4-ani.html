TM - 39 TDT - Editie Aniversara de 4 ani :)

joi, 3 decembrie 2015

Dragi prieteni ai Taberei de Testare, am ajuns si in acest moment cand facem 4 ani de Tabara de Testare.

Va multumim ca ati fost alaturi de noi, tot acest timp, si va invitam sa sarbatorim impreuna inca o editie aniversara.

Gazda noastra in acest an sunt cei de la Unified Post. Pe langa surprizele aniversare va pregatim urmatoarele topicuri:

1) Security Testing (Adela Neczpal)

- The importance of security testing.

- Security testing types.

- Are we secure ?  How much security testing YOU DO ?

- Security testing software tools.

- Starting directions in developing security testing.

- Advantages of secure applications.

2) Load Testing using Jmeter (Sabin Uiv ăroșan)

- JMeter - general introduction

- Most used JMeter components and controls

- Use cases and samples

- JMeter as automation and scripting tool

- JMeter for volume testing and UP concrete usage

3) Page Object Model (RSpec/Capybara) (Florin Semenescu)

- Page Object Model pros & cons

- Rspec overview

- Capybara overview

- SitePrism overview

- Example
