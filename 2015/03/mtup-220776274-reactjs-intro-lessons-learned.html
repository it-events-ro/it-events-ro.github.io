ReactJS: Intro & Lessons Learned

miercuri, 4 martie 2015

Hello everyone! It has been almost a month; time to get together talk about something interesting again.

Andrei Cimpean will give us an introduction to ReactJS and how it changes building single page applications. We're going to discuss about transpiling code to JavaScript, React's virtual DOM and various things learned while building components.

After the presentation we'll also have a workshop led by Andrei - "Let's build some React components".

Details about what you will need to participage in the workshop will follow.
