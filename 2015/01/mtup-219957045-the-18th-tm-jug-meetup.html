The 18th TM-JUG meetup

joi, 22 ianuarie 2015

Hello, everyone!


We would like to invite you to an ad-hoc TM-JUG meetup, where Sergi Almar will present us the Spring WebSocket framework, in a presentation called ' Going real time with Spring WebSocket'

Abstract:

WebSocket support has been part of Spring since 4.0. In this session we'll review the architecture behind it and we'll try to show how to build highly scalable websocket based applications. We will also cover the new features in Spring 4.1, the new Spring Security 4 support for WebSockets.
