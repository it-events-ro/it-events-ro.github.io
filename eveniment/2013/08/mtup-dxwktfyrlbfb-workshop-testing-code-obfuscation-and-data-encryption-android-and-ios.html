Workshop: Testing Code Obfuscation and Data Encryption. Android and iOS

sâmbătă, 3 august 2013

[New info for the attendees! Please be a good tester and read all the description!]

Hi all,


In the next session (August 3rd), as a content owner, I plan for a small workshop where we'll talk about some mobile security testing ; mostly code obfuscation and data encryption.


With just a few easy steps you can access an application code and see its logic and more important for us testers, you can see if the database is encrypted and what kind of information is available to "the naked eye".
