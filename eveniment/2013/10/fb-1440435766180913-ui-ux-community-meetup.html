UI/UX Community Meetup

joi, 10 octombrie 2013

UI/UX Meetup are ca si scop reunirea comunitatii de developeri si desgineri din mediul IT clujean. Aceasta comunitate este destinata Web si Mobile Designerilor, Developerilor, Architectilor, Expertilor pe Usability, Experience Designerilor, etc. - oricui implicat in design-ul, dezvoltarea, programarea si implementarea web site-urilor si aplicatiilor mobile indiferent de platforma. La aceste intalniri periodice vom interactiona cu profesionisti din industrie personal cat si prin prezentari; vom compara ultimele tool-uri, trenduri, teorii, limbaje, template-uri; vom face schimb de know-how, tips & tricks. Vom avea frecvent invitati din zona de Web and Mobile ca si invitati sau ca speakeri. Ca si prim invitat al meetup ului il anuntam pe Mircea Botez - Software Arhitect care va participa la http://uxmasterclass.com/2013/ in Roma si care ne va aduce impresii si opinii de ultima ora din lumea optimizarii experientei si interactiunii cu utilizatorul. UX, sau User Experience reprezinta procesul de optimizare a paginii, prin ochii utilizatorului. Multe persoane intampina dificultati in utilizarea unor site-uri / aplicatii care ar trebui sa fie usor de folosit. UI, sau User Interface, inseamna tot ceea ce utilizatorul tau vede atunci cand iti acceseaza aplicatia. Un designer UI, de cele mai multe ori se ocupa de partea de dezvoltare a interfetei aplicatiei, insa scopul final al acestuia, este acelasi utilizator. De aceea colaborarea dintre un designer UI si unul UX este absolut necesara pentru tinta finala a aplicatiei sau a paginii tale web.
