AIO Networking Workshop - Implementare Retea Microsoft

luni, 3 septembrie 2012

ZIUA 1 Instaleaza Reteaua Fa-ti singur cablul de retea Instaleaza Switch-ul Fast sau Giga Ethernet? Adrese IPv6, IPv4 si MAC Deseaneaza Diagrama Retelei Testeaza Retea (arp & ping) ZIUA 2 Conecteaza Reteaua Instaleaza Router-ul Conecteaza Retelele cu Rute Statice Instaleaza Serverului de Rutare si VPN Conecteaza Retelele cu Rute Dinamice Conecteaza Retelele prin VPN Testeaza Rutele (route, tracert) ZIUA 3 Adauga Functionalitate Instaleaza DHCP Server Instaleaza DNS Server Instaleaza HTTP Server Protocoale (SMTP, POP3, IMAP4, FTP) Testeaza Protocoale (telnet, nslookup) ZIUA 4 Securizeaza Reteaua Cum functioneaza un Firewall? Configureaza Firewall de Retea Amenintari de Securitate la Retea Testeaza Securitatea Retelei ZIUA 5 Proiect pe Echipe Instaleaza reteua ta interna si conecteaza-te cu celelalte retele cu ajutorul VPN-ul din Microsoft RRAS.
