ITCamp Community Event | Timisoara

sâmbătă, 2 iulie 2016

În această vară, comunitatea ITCamp în parteneriat cu Microsoft România organizează un nou eveniment pentru profesioniștii IT din Timișoara. Evenimentul va avea loc pe data de 2 iulie 2016, la Hotel Timișoara. Participarea este gratuită în limita locurilor disponibile. Înscrierea, aici: http://www.eventbrite.com/e/itcamp-community-event-timisoara-2-july-2016-tickets-25861120307 Agenda: 8.30 - 9.00 - Registration 9.00 - 9.45 - Mobile Apps and Mobile Engagement [Mihai Tataran] Level 300 Microsoft Azure has provided a technology called Mobile Services for a few years, which is great for creating Mobile enterprise solutions with backends in the cloud, for all major mobile platforms. Nowadays the Mobile Apps set of features is becoming richer at a fantastic pace. Not only you can create cloud backends for mobile apps, but also you can: autoscale them, integrate them with your on premises solutions, work offline. One of the coolest features in this spectrum is Mobile Engagement which promises to increase app usage and user retention through real-time analytics, measurements, predictions about their behavior. Let us see them altogether in a demo-driven presentation. 9.45 - 10.30 - Unleashing the power of Azure SQL Data Warehouse[Ciprian Jichici] | Level 300 Historically, data warehouses have required fixed combinations of storage and compute, often under-utilizing expensive resources. With Azure SQL Data Warehouse, storage and compute scale independently. You can dynamically deploy, grow, shrink, and even pause compute, taking advantage of best-in-class price/performance. Also, SQL Data Warehouse uses the power and familiarity of T-SQL to let you easily integrate query results across relational data in your data warehouse and non-relational data in Azure blob storage. In this session we will take a deep dive into the inner working of the Azure SQL Data Warehouse. We will talk about delivering breakthrough query performance, combining relational and non-relational data in queries, and implementing hybrid scenarios with an enterprise-class SQL Server experience. Also, we will talk about real-life examples taken from the current Azure SQL Data Warehouse projects we are working on. 10.30 - 10.45 - Cofee Break 10.45 - 11.30 - 2016, a new era of OS and Cloud security [Tudor Damian] Level 200 The global security landscape is changing, now more than ever. With cloud computing gaining momentum and advanced persistent threats becoming a common occurrence, the industry is taking a more focused and serious approach when it comes to security, especially after some of last years’ heavily publicized incidents. Join this session for a discussion on what Microsoft is doing to protect against these new security threats with fresh approaches taken both at the server & client OS level, as well as in Azure. 11.30 - 12.15 - Enterprise Mobile Device Management Using Microsoft Intune and System Center Configuration Manager [Radu Tomoioaga] | Level 200 Manage Windows 10 devices via MDM with on-premises infrastructure. Advantages & Disadvantages with on-premises management. Prerequisites to enable On-Premises Mobile Device Management. High Level Scenarios 12.15 - 12.30 - Break 12.30 - 13.15 - Azure Microservices in Practice [Radu Vunvulea] | Level 300 In this session we will take a look on Azure Service Fabric and how we can create a web application using ASP.NET Core and microservices. During the presentation we will create a web application that will be hosted in Azure Service Fabric and use the most important capabilities of Azure Service Fabric.
