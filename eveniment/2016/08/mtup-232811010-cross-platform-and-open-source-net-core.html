Cross-Platform and Open Source - .NET Core

In the context of Open Source and Cross-Platform applications and languages, Microsoft needed to adapt and to evolve. This is an introduction session to .NET Core and what can we do with it.

About the speaker: Lucian Ziman
Passionate about technology, Lucian Ziman is a .Net Developer in SDL Cluj, part of a team responsible for technical design and implementation of the world's #1 translation software. In the end, they’ll make 'SDL Trados Studio' part of the everyday lives of many, many people, using latest technologies on the market.
