Transylvania Java User Group meeting #55: Low level details for java developers

joi, 11 august 2016

Transylvania Java User Group meeting #55:

Event title: "Low level details for high level developers"

Abstract:
We're using a lot of abstractions as developers to make our
lives easier. Nobody remembers what the opcode 0xEB does - nor should
they! Still, knowing a little bit about the abstractions can help us
make better use of the hardware/software stack we use, integrate with
components written in other languages and track down difficult to find
issues.

Agenda:

- A refresher about the x64 architecture
- From java source code to machine code and back
- A couple of performance hints
- Efficient memory access patterns
- Low-level tools for performance analysis

Speaker bio :

Attila is a polyglot software craftsman who spent the last two years optimizing a big data system for gene analysis and wants to share the lessons learned.

Speaker blog: https://www.grey-panther.net/

Event timelines:

18:30 - 19:00 reception

19:00 - 20:00 presentation

20:00 - 20:30 Q&A session
