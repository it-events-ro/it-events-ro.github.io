Global Azure Bootcamp Cluj-Napoca | ITCamp Community

sâmbătă, 16 aprilie 2016

Acesta este al patrulea an când organizăm Global Azure Boot Camp. Acesta este un eveniment la nivel global care are loc în peste 136 de locații. Ca și anul trecut, Clujul nu se lasă mai prejos și apare pe harta Azure. Pe data de 16 Aprilie vă invităm pe toți la acest eveniment din Cluj-Napoca, care v-a conține 3 workshop-uri.


Participarea la eveniment este GRATUITĂ, așa cum a fost și până acum la orice eveniment organizat de comunitatea ITCamp. Vă puteți înscrie aici sau pe pagina următoare http://itcamp-community-2016-04.eventbrite.com .
Pe data de 16 Aprilie ne propunem să avem 3 workshop-uri de câte 90 de minute fiecare, unde putem să învățăm împreună cum să folosim diferite servicii Azure. Fiecare workshop conține o parte teoretică și una practică. Din această cauză, o să aveți nevoie de un laptop.


De ce aveți nevoie:
- Laptop
- Visual Studio 2013 sau Visual Studio 2015
- SDK de Microsoft Azure https://azure.microsoft.com/en-us/blog/announcing-the-azure-sdk-2-8-for-net/
- Un cont gratuit care poate să fie făcut folosind următorea adresă http://www.windowsazure.com/pricing/free-trial/ (sau <a>www.windowsazure.com)</a>

Dacă doriți puteți să vă grupați în grupuri de 2-3 persoane la același laptop.

Program:
09:00-09:30 - Sosirea participanților
09:30-11:00 - Azure Service Fabric in Action (Radu Vunvulea)
11:00-12:30 - Service Bus hands-on (Radu Pascal)
12:30-14:00 -Azure SQL Database - Everyone's intelligent cloud database (Silviu Niculiță)

Mulțumim sponsorilor pentru susținere, ISDC și Small Footprint!
