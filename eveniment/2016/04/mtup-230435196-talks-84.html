miercuri, 20 aprilie 2016

Although the weather is moody, we keep on going, looking forward to a new Talks edition. Don't forget to RSVP!

Here’s the agenda for Talks #84:

19:00 - 19:15  - Welcome!


19:15 - 19:45 - Andrei Pietrosu  → Litespeed Web server - Sometimes commercial is great!

LiteSpeed Web Server (LSWS) is a high-performance Apache drop-in replacement. LSWS is the 4th most popular web server on the internet and the #1 commercial web server. Although the best version is the paid subscription, the open source one is pretty decent. This presentation will cover Litespeed internals, "the point and shoot" design, great add-ons, benchmarks and how and why I became a big fan of it :).


19:45 - 19:50 - Break


19:50 - 20:20 -  Alexandru Badiu → Open Source Licenses and how to choose one

This presentation will introduce you to the major open source licenses available, the key differences between them and give you information about which one is right for you. We’ll also talk about copyright, public domain, how not to infringe licenses and tips and tricks about maintaining an open source project.


20:20  – Q&A - Networking! :D
