ES2015 crash course

joi, 14 aprilie 2016

Greeting clu js ers,


We are very happy to announce a new meetup in April, during which we will learn more about the modern JavaScript syntax described by EcmaScript2015 (previously known as ES6). We will have two presentations, approximately 30 minutes each and then another 30 minutes for open talks on the subject.

----------

First Talk : Alex Moldovan - Full Stack Developer @ Fortech

About the speaker: Alex is very passionate about both JavaScript and Ruby and recently has been working a lot with ES2015 in his Angular and React projects. He also secretly hopes that functional programming will take over the world, so he is preparing the best he can for this.

About the presentation: In the first presentation, we will learn about the basics of the new JavaScript syntax with code examples. The main focus will be around the syntax that can be used to make the code cleaner and more expressive: arrow functions, destructuring, function specific operators (default, rest, spread) and many more. Also, we will talk about how you can integrate the new syntax into a project by using babel.js

Presentation level : Beginner - Intermediate

----------

Second Talk : Gabriel Imre - Full Stack Developer @ Endava

About the speaker : Gabriel is a JavaScript fanatic with more than 8 years of combined experience with client and server side JavaScript, working on various projects starting from vanilla js to Angular/React to Node.js.

About the presentation: In the second part of the meetup, we will learn more about asynchronous programming and asynchronous patterns with ES2015 generators and promises. We will see how you can leverage the full power of the new features on the server, with a case study on Koa.js . Koa is a new web framework designed by the team behind Express , which aims to be a smaller, more expressive, and more robust foundation for web applications and APIs.

Presentation level : Intermediate - Advanced
