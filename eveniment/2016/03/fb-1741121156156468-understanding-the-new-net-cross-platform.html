Understanding the new .NET cross-platform

joi, 24 martie 2016

Ești developer ASP.NET, dar te-ai lovit de costurile de găzduire, licențiere și mentenanță? După mulți ani de dezvoltare privată a framework-ului .NET, Microsoft lansează în comunitatea open-source .NET Core, o implementare modulară, multi-platformă și modernă a clasicului .NET. În cadrul acestei prezentări vom studia părțile componente ale noului ecosistem prin testarea unei aplicații ASP.NET Core scrisă în C#, complet compilată și “găzduită” pe Linux. Agenda evenimentului: - Introducere în ecosistem:.NET Core vs .NET Framework 4.6; - Noul compilator Roslyn; - De la cod intermediar (Intermediate Language) la cod nativ (.NET Native); - Aplicații universale (rulează pe orice platformă sau dispozitiv): ASP.NET Core și Universal Windows Platform - Utilitare: .NET Core Command Line Interface și Visual Studio + Visual Studio Code; - Microsoft + Xamarin = iOS && Android; - Instalare .NET Core pe Ubuntu; - Demo: compilare și testare aplicație ASP.NET în Linux. Invitatul nostru: Ionuț Dănilă lucrează ca Windows Platform Developer la MiOS, iar în paralel este trainer .NET în cadrul comunității tehnice IT-st și colaborator al Facultății de Informatică Iași. Se autocaracterizează ca fiind organizat, perfecționist, punctual și energic, probabil acesta este secretul pentru a face față cu brio tuturor activităților zilnice.
