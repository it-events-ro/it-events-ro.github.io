Selenium Unconference II

sâmbătă, 17 mai 2014

The Unconference Selenium Unconference is a volunteer-run, non-profit event organised by members of the community that aims to bring together users and developers to share ideas, socialize and share their expirience. The Unconference aims to bring Selenium closer to the Cluj-Napoca community, presenting how it is used by the big players in the industry and how we as a team can contribute to the Selenium project as it is an open source project. The second edition will continue our effort to build a confortable place to talk about Selenium both for the newcomers to this field and also for those working now for several years with it. After the presentations we encourage people to share issues/problems they've met while working with Selenium, underlining the unconference concept. Selenium Selenium is an open-source project for testing web-based applications in any web browser on any operating system on any device desktop or mobile. The Organizers If you have any questions about the conference, please contact with the organizers and someone will contact you! Speakers Dave Hunt works from home for Mozilla, where he assists teams to create automated tests for their projects - ranging from Mozilla's web properties to the Firefox web browser and Firefox OS for smart phones. Dave's talk will be an introduction to ZAP (Zed Attack Proxy) for web security testing, and a demonstration of how you can integrate it into an continuous integration system with automated functional tests using Selenium WebDriver, py.test and plugins. Costin Ion is a Quality Engineer Developer working for Adobe Systems where he has been developing tools for automated testing and developer productivity since 2008. He has over 10 years of test engineering experience in different areas including software security, eCommerce, web services, analytics, mobile technologies. Costin's talk will be about how Adobe created their test automation solution for a key component of Digital Publishing Suite (http://www.adobe.com/products/digital-publishing-suite-enterprise.html). The presentation will cover topics from different areas like cross mobile platform automation, web automation practices and tools into mobile world, test approach for complex architectures, testing analytics in mobile context.
