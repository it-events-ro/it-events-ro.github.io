Joint Workshop: Java & HTML5 based cross-platform mobile development framework

joi, 15 mai 2014

Our friends from TM-JUG have developed together with Oracle Romania an interesting event concept centered on a presentation of a Java & HTML5-based cross-platform mobile development framework, that will help in creation of Android and iOS apps, using a common code-base.

http://lh4.ggpht.com/-s0Jr6KL3jQw/UljbOfR-t6I/AAAAAAAACCU/yKz_ToOSJiM/Oracle%252520ADF%252520Mobile%252520_thumb%25255B2%25255D.jpg?imgmax=800

Timisoara Mobile Development Group joins this project encouraging its member to attend this co-branded project that also counts for as group's May meetup.

Here you may find the original presentation:


“Oracle has published a Java based mobile development framework, which - at the moment - is capable of deploying the same code-base to Android and iOS operating systems. For Java developers that's great news, as now we can reuse our existing Java skills to develop enterprise mobile applications.

In this session, Jernej Kaše , Oracle FMW Specialist based in Slovenia, will describe the mobile framework architecture and show some cool demos - from hello world, to how to use your phone camera with Java and even remote data synchronisation. The most persistent participants can stay a few hours extra and do a hands-on jam session. They will learn some key concepts of cross-platform mobile development:

·  ADF Mobile Architecture

·  Setting up development environment

·  ADF Mobile application overview (feature, content, page, task-flow)

·  Develop a hello world sample

·  Using device features (sending SMS, using camera,…)

·  Using Java within ADF Mobile

·  Building a simple CRUD application

Those interesting to stay for the hands-on session should make sure their laptops have at least 3GB RAM, JDeveloper[masked] installed and extended for ADF Mobile (Help->Check for updates), an Android device, Android SDK and API level suitable for their respective device OR Apple Mac, Xcode 4 (not 5!) and iPhone Simulator.
