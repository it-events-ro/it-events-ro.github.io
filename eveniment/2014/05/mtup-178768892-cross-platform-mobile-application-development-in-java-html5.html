Cross-platform mobile application development in Java & HTML5

joi, 15 mai 2014

Hello!


Oracle Romania and TM-JUG invite you to a presentation of a Java based cross-platform mobile development framework, which builds Android and iOS apps using the same code-base.

Further details:

“Oracle has published a Java based mobile development framework, which - at the moment - is capable of deploying the same code-base to Android and iOS operating systems. For Java developers that's great news, as now we can reuse our existing Java skills to develop enterprise mobile applications.


In this session, Jernej Kaše, Oracle FMW Specialist, will describe the mobile framework architecture and show some cool demos - from hello world, to how to use your phone camera with Java and even remote data synchronisation. The most persistent participants can stay a few hours extra and do a hands-on jam session. They will learn some key concepts of cross-platform mobile development:

·  ADF Mobile Architecture

·  Setting up development environment

·  ADF Mobile application overview (feature, content, page, task-flow)

·  Develop a hello world sample

·  Using device features (sending SMS, using camera,…)

·  Using Java within ADF Mobile

·  Building a simple CRUD application


Technical requirements:

- Laptop with at least 3GB RAM

- JDeveloper[masked] installed and extended for ADF Mobile (Help->Check for updates)

- Android device, Android SDK and API level suitable for your device OR Apple Mac, Xcode 4 (not 5!) and iPhone Simulator”
