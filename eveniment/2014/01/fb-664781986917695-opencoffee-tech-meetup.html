OpenCoffee Tech Meetup

joi, 23 ianuarie 2014

Hope you've had a great Christmas time! As we are still celebrating, OpenCoffee group has come up with a gift as well - a tech meetup. We want to meet each of you in a pleasant atmosphere, to present the idea that has started this group - and hope you'll embrace it yourself as well! Besides this, we also want to analyze together the top web technologies/frameworks that helped others to build great products and that are worth to be presented during this year. In the same conjuncture, we will have free/open discussions about web/mobile tech scene and find inspiration by showcasing the latest web products that have been created lately. Creative ideas & great knowledge sharing to fuel your passion for programming and startups should be the result of this meetup! Let's make it happen! First meetup agenda: 19:00-19:20 : Welcome guests, present the idea of this tech meetup 19:25-20:00 : Benefits Groovy and Grails web framework brings to SpotTune 20:00-20:30 : Discussions and planning of technologies, frameworks, techniques and tools worth to be presented this year.
