Continuous Delivery cu Java, AWS & Jenkins

miercuri, 26 noiembrie 2014

Dragi programatoare, Girls Who Code Romania si Rinf vă invită să luați parte la o prezentare cu demo interactiv de Continuous Delivery. Sorin Jucovschi, director tehnic la RINF, vă va arăta cum puteți folosi Amazon Web Services și Jenkins astfel încât după fiecare commit aplicația dezvoltată să fie întotdeauna gata de livrare! Știi Java și vrei să afli mai multe despre cum îți poți îmbunătăți timpul de dezvoltare? Aplică acum! Participarea este gratuită, dar locurile sunt limitate! Înscrie-te aici până miercuri, 26 noiembrie, ora 12:00: http://goo.gl/f6l0m5 Continuous Delivery este tehnica folosită pentru a asigura faptul că dupa fiecare commit în aplicație, produsul software rezultat va fi stabil și gata de livrat, direct la client. Amazon Web Services reprezintă o colecție de servicii web, păstrate în Cloud, puse la dispoziție de către Amazon. Jenkins este o aplicație open source, scrisă în Java, care ofera servicii de integrare continuă (Continuous Integration).
