Page Object Pattern abstraction over Selenium by Robert Biter

miercuri, 3 decembrie 2014

In this meeting Robert Biter (senior java developer) will be presenting an open source library - a page object pattern abstraction over Selenium Webdriver that he has put together for community usage. The goals he had in mind when writing this library were:

·  to be very easy to use

·  have a low learning curve

·  be easy to extend with custom behavior

What to expect from his presentation:

·  See all core components presented with examples

·  See a live implementation of a test scenario from scratch

·  Q&A session
