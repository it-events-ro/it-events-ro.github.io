Conference.py #1

sâmbătă, 18 octombrie 2014

Toamna este un anotimp al începutului, fiind vremea unui nou an școlar, unui nou început de an universitar, precum și sezonul în care vacanțele lungi sau scurte, plănuite sau inopinate, lasă locul altor activități. Este anotimpul în care începe stagiunea la operă și în care studenții revin timizi în orașul studenției lor, tânjind după aventurile verii proaspăt trecute. Toamna este și anotimpul conferințelor IT consacrate, atât pe plan local, cât și la nivel național, menționând Design Jam, DefCamp sau binecunoscutul CodeCamp. Grupul RoPython pune pe harta evenimentelor IT prima conferință dedicată limbajului Python din România, Conference.py #1, ce se va desfășura în Iași, pe data de 18 octombrie. Conferința va face parte dintr-un eveniment mai larg, denumit "Python Weekend", ce va mai consta dintr-o serie de workshopuri, desfășurate în ziua imediat următoare conferinței. Grupul RoPython s-a format în urmă cu șase luni, într-un loc înconjurat de piscuri înzăpezite, din dorința de a promova limbajul Python pe plan local și regional. Format din oameni entuziaști, experimentați în limbajul Python și cu rădăcini în proiecte open source de anvergură, grupul își propune nu doar promovarea limbajului în România, ci și deschiderea apetitului tinerilor pentru programare, tehnologie și cunoaștere, dorindu-și totodată pentru Python o expunere mai mare în liceele și facultățile din țară. Considerăm că principiile ce stau la baza limbajului, principii evidențiate în slove de către cunoscutul dezvoltator Tim Peters, în al său Zen of Python [1], pot fi aplicate și în viața de zi cu zi, ele putând crea nu doar dezvoltatori software cu o viziune clară și un set de principii după care să ghideze evoluția programelor, ci și oameni echilibrați și organizați. Cu o istorie de peste douăzeci de ani, limbajul Python a evoluat, în decursul timpului, de la un simplu limbaj de programare la un mediu vibrant, ce pune la dispoziția fiecăruia toate resursele necesare pentru transpunerea ideilor în realitate, înconjurat totodată de una dintre cele mai primitoare și cuprinzătoare comunități tehnice din lume. Având la activ câteva întâlniri și workshopuri, RoPython își propune trecerea la următorul nivel, organizarea unei conferințe, care să adune la un loc toți dezvoltatorii pasionați de limbaje interpretate, open source și tehnologie, precum și cei neinițiați, ce doresc să facă primii pași în descoperirea unui nou limbaj de programare. Conference.py propune o zi întreagă de prezentări tehnice, venite din partea unor speakeri implicați în proiecte open source de anvergură, cu subiecte ce vor purta participanții prin cloud computing folosind Python, in(tro)ducere în metaclase și sisteme de fișiere bazate pe Git, cuvintele cheie ale evenimentului fiind open-source și bineînțeles, Python. Vrem ca această conferință să rezoneze cu publicul pasionat de IT din Iași și nu numai, astfel încât prin ea să reușim să formăm un mediu propice dezvoltării comunității iubitorilor de Python și pasionaților de tehnologie din România. Reunind speakeri din aproape toate colțurile țării, prezentările din cadrul conferinței se anunță a fi incitante, oferind o privire largă asupra locurilor unde Python poate fi folosit. Următorii speakeri vor lua parte la prima ediție Conference.py, sperând ca de la an la an să creștem tot mai mult numărul lor și al prezentărilor. * Vlad Temian (Timișoara) - backend developer la PressLabs * Ionel Cristian Mărieș (Cluj) - software consultant * Alessandro Pilotti (Timișoara) - CEO și fondator Cloudbase Solutions * Alex Morega (București) - web developer * Gabriel Samfira (Timișoara) - senior cloud engineer la Cloudbase Solutions În ziua imediat următoare conferinței, se vor desfășura o serie de workshopuri, în cadrul Facultății de Informatică, susținute de mentori din cadrul grupului RoPython și nu numai, subiectele variind de la introducere în limbajul Python, programare funcțională și cloud computing. De asemenea, Conference.py este primul eveniment IT din Iași, listat în cadrul inițiativei europene Code Week, program ce își propune demistificarea programării și atragerea tinerilor către această îndeletnicire. Ne dorim un număr cât mai mare de participanți, care să ia parte la aceste evenimente. Grupul RoPython vă așteaptă cu drag la prima ediție Conference.py și prima ediție Python Weekend.
