Software craftsmanship: Autumn code retreat

sâmbătă, 20 noiembrie 2010

As software development is maturing as a discipline, you need to master many techniques in order to do your best. A great way to learn is during code retreats, which are facilitated, hands-on gatherings, where programmers try to solve various coding challenges, share their experiences and learn from each other, while having fun.

2009 saw the first code retreat in Timisoara ( http://www.meetup.com ..., and people have been asking for another one. We are glad to oblige, so this Saturday (20 Nov 2010) we'll be organizing a new one.

Our host will be Alexandru Bolboacă ( http://www.alexbolboa ..., one of the most experienced practitioners of agile software development in Romania, having worked alongside very experienced programmers such as J.B. Rainsberger or Corey Haines.

Techniques we'll cover? The focus is on TDD, emergent design (OOD), refactoring and unit testing, but feel free to bring your own issues and we'll discuss them.

To fully benefit from the session, you are kindly asked to bring a laptop with the preferred development environment (vim fans get front row seats), plus (optionally) a pairing partner, as pair programming is the norm. Don't worry if you don't have one though, as partner switching is encouraged. The coding language is left at your choice, so Java, C#, C++, Ruby, PHP or Python are more than ok.

The event starts at 09:30am, and shouldn't last more than 8 hours, unless we'll have too much fun. As the seating is limited (16 places), please make sure to confirm your presence in advance.

See you there!
