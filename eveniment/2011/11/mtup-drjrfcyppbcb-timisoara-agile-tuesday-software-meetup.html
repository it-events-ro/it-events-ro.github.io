Timisoara Agile Tuesday Software Meetup

marți, 1 noiembrie 2011

This event, the first event of the new Agile Tuesday schedule will be designated to be a brainstorming session for topics in the upcoming Agile Meetups.

We will slice all Agile methodologies and select what topics we should address in the meetups that follow. Also, we'll create a list with members that would like to present something at one of the future meetups.

Program of the event:

- 7:30 registration;

- 7:45 introducing ourselves;

- 8:00 Agile methodologies slice-up;

- 8:20 small 5 minute break;

- 8:25 Topics proposals;

- 8:55 meetup retrospective.

Anyone can propose topics.

Event venue is the Timisoara Software Business Incubator, on 9th Republicii blvd., Timisoara, 3rd floor, room 310-311.

Link: http://www.meetup.com/The-Timisoara-Agile-Software-Meetup-Group/events/38472762/
