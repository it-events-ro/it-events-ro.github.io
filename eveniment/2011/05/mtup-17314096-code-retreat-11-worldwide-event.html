Code Retreat #11 - worldwide event

sâmbătă, 7 mai 2011

Are you a passionate software developer? Ever wanted to go back to the days when you could just work on your coding ideas without having any pressure? Do you want to try out modular design, pair programming, automatic tests, Test Driven Development, various programming languages, refactoring, clean code etc.?

Then this day is for you.

Join Alexandru Bolboaca and a self-selecting group of people who want to learn just as much as you do. Share your passion and your knowledge, learn from others and leave happier at the end of the day. A Code Retreat is the best way of acquiring new coding and design skills in only one day, for free. If you've never attended a code retreat, find out how it works here .

This one is special because other ones will take place around the world on the same day and we will connect with the other groups.

Mosaic Works is sponsoring lunch for this event.
