Code Retreat #4 - worldwide event

sâmbătă, 7 mai 2011

Clean code. Unit testing. TDD (Test Driven Development). Refactoring. Pair programming. Object Oriented Design. SOLID principles.

If you would like to practice (or find out more about) any of the above, improve your skills and see how other developers approach coding, then you should come to the fourth code retreat in Timisoara on May 7th, 2011.

The programming language is not restricted, so feel free to use any language and impress us with your expert ninja skills.

Be warned: on that Saturday you will be part of a global movement promoting coding excellence and software craftsmanship. Code Retreats will be happening all over the globe ( http://coderetreat.ning.com/events/event/listByDate?date=2011-05-07 ), and we'll probably get in touch with other locations during the day.

If you've never attended a code retreat, find out how it works here . The event will last for approximately 8 hours.

Link: http://www.meetup.com/The-Timisoara-Agile-Software-Meetup-Group/events/17370457/
