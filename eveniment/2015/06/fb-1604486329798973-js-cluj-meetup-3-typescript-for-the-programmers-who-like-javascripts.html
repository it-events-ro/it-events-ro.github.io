JS Cluj Meetup #3 – Typescript for the programmers who like JavascriptS

luni, 22 iunie 2015

Are you a JS dev interested in the latest trends? Then join us at the 3rd meetup of JS Cluj Community on June 22. # Agenda: 6:15 – Registration. Put on a nametag! 6:30 – Event starts with welcome from the organizers 6:40 – Key Note Presentation: Andrei Sebastian Cimpean - Typescript for the programmers who like Javascript Transpilation is a trending process in the Javascript world. One of the new languages that's taking advantage of this is Typescript. Built by Microsoft and targeted mostly towards C# or Java programmers, it grew a lot in the latest months and is now replacing AtScript as the language used to build AngularJS 2. At this meetup we'll compare Typescript with Javascript and discuss about different ways to use it in the browser or with nodejs. 7:40 – Q&A 7:50 –. Networking # About the speaker: • Andrei Sebastian Cimpean is a Software engineer and Team Lead at Softvision. He is passionate about software and lately he delivered project in which the part he was involved was written in Typescript, BackboneJS and ReactJS. Registration on Meetup is mandatory for this event!
