Taking Lambdas and Streams to the Next Level With JDK8

miercuri, 23 septembrie 2015

Join us and our partners to a new great Java event together. Special guest: Simon Ritter, Head of Java Evangelism at Oracle. This talk will start with a brief introduction to the concepts of Lambdas and Streams and then delve into how to really get the most from them. We'll focus on how developers need to think differently using lambdas and streams to write code that is functional in nature, rather than imperative. This is a big change for Java developers and delivers a much simpler way to utilise the power of multi-core, multi-processor hardware. Come and find out how to exploit JDK 8 to its full potential!
