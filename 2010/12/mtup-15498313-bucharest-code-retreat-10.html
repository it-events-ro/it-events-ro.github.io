Bucharest Code Retreat #10

sâmbătă, 11 decembrie 2010

We are happy to announce a new code retreat - the 10th held in Bucharest. Come and practice your developer skills, expand your horizon while having a lot of fun.

All code retreats are free events, and this is not an exception.

What is a Code Retreat?
A day for coding without deadlines. A day to get intimate with your code and forget the management pressure.

How do we do it?
We gather a bunch of people, group in pairs, settle on a problem and work on it for six sessions, each session lasting 50 minutes. We take 10 minutes breaks between sessions, and a longer break between the 3rd and the 4th session. Also, we change pairs in the breaks and delete all the code we've written in that session (it will be fun, you'll see).

Programming language does not matter. Usually, there are at least two people that know a particular language. But in case you want to code in a peculiar language, then maybe you should bring yourself the programming pair.

Be sure to bring a laptop and prepare yourself for getting into the zone!

Rules!
The fundamental rules in a Code Retreat are to do it in pair, have fun and don't get the problem solved (we mean it!) We focus on exploring new ways to express the problem in code instead of actually solving it.

Lunch

We will order food from a catering service, chosen based on the participant's preferences.
