joi, 28 iulie 2011

Javascript saga, part 1

Prezinta: Istvan Hoka

Javascript a cunoscut o renastere in ultima vreme. Incepe sa fie din ce in ce mai folosit pe partea de client, pentru a implementa aplicatii complexe care ruleaza in browser, si incepe sa fie folosit si pe partea de server.

Javascript e un limbaj functional puternic, insa cu niste probleme serioase de design. Din fericire, daca cunoastem destul de bine limbajul, ne putem feri de problemele acestea si sa ne folosim doar de partile bune ale limbajului.

Javascript: The Good Parts este o carte scrisa de Douglas Crockford care arata partile bune si partile rele ale limbajului.

Voi incepe o serie de 3 prezentari pe aceasta tema -- limbajul Javascript:

Javascript saga, part 1 - prezentare despre structura limbajului Javascript si modelul obiectual, cu accent pe lucrurile care il fac diferit de alte limbaje si pe problemele de design. Javascript saga, part 2 - prezentare despre functiile din Javascript. Functiile in Javascript sunt cea mai buna parte din limbaj, oferindu-i o mare putere expresiva, comparabila si chiar depasind-o pe ce a a limbajului de programare Ruby. Javascript sava, part 3 - Coffeescript. Coffeescript este un meta-limbaj, care se compileaza in Javascript. Scopul lui este sa ofere un mod mai frumos de a folosi partile bune din Javascript. Ruby Koans
