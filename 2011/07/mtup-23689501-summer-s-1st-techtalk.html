Summer's 1st TechTalk

joi, 14 iulie 2011

This meetup will be hosted by Evozon Systems, which will provide us the infrastructure and other things that we need for such an event.

Disclaimer: This is a Cluj.rb Meetup. The opinions expressed here represent the speaker's opinion and not those of the hosting company e.g. Evozon Systems.

Code smells

Most of the people focus on how to write clean or SOLID code. Hence the approach is how to do it. I present another approach that will focus on how to recognize bad written code.

:time => 20-30 min

:speaker => Andrei, andreiursan.com

Clojure on AWS EC2

This language has quite a momentum, therefore I've decided to see how complicated or simple is to get a web app written in clojure and hosted on EC2.  I will present my personal experience and opinions.

:time => 20-30 min

:speaker => Andrei, andreiursan.com
